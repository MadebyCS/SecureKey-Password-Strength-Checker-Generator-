# SecureKey-Password-Strength-Checker-Generator:
A browser-based password security tool built with HTML, CSS, and JavaScript. Designed to help users evaluate the strength of their passwords and generate cryptographically secure ones instantly.
Features
Password Strength Checker — Analyses passwords in real-time based on length, uppercase letters, lowercase letters, numbers, and special symbols. Classifies passwords as Weak, Medium, or Strong with a visual strength bar.
Password Generator — Generates secure random passwords using the browser's built-in window.crypto API. Supports customizable length (6–32 characters) and character type toggles (Uppercase, Lowercase, Numbers, Symbols).
Show/Hide Toggle — Toggle password visibility while typing for ease of use.
Copy to Clipboard — One-click copy button for both the checker input and the generated password, with toast notification feedback.
# Tech Stack:
Built using HTML5, CSS3, and Vanilla JavaScript. No frameworks, no libraries, no dependencies — runs entirely in the browser.
# How to Run:
Download or clone the project. Open the folder in VS Code. Click Go Live (Live Server extension) on the securekey.html file. The app opens instantly in your browser.
